# C++ Reference

Show a floating holographic code panel.

Use authentic modern C++.

Example:

```cpp
#include <array>
#include <cstdint>

class Blink
{
public:
    void run()
    {
        while(true)
        {
            gpio::write(LED_PIN, true);
            timer::delay_ms(500);

            gpio::write(LED_PIN, false);
            timer::delay_ms(500);
        }
    }

private:
    static constexpr uint8_t LED_PIN = 25;
};

int main()
{
    Blink app;
    app.run();
}
```
