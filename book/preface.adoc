= Preface

This book is a practical guide to modern embedded software development, aided by microCI.

It focuses on the ideas and practices that matter across embedded projects: firmware structure, toolchains, peripherals, timing, testing, and the habits that make systems easier to maintain over time.

microCI is part of the story because it helps make the workflow reproducible and scalable, from early bring-up to more complete development and testing pipelines.

The Raspberry Pi Pico family is the platform used throughout the examples so the reader can move from basic bring-up to more complete workflows without changing the core hardware.

The main examples use modern C++, with MicroPython appearing where it helps clarify a concept or shorten an example.
